# MDK API Javadocs

These are the Javadocs for the Solace Micro-Integration Development Kit (MDK) framework.

Each folder here contains the Javadocs for a specific module of the MDK. They provide detailed information about the classes, methods, and interfaces available in each of the MDK's modules.

For general guidance on using the MDK, refer to the [Development Guide](../Micro-Integration%20Framework%20Development%20Guide.pdf) and the [Samples](../samples/README.md) for practical examples.

## Modules

- [Micro-Integration Framework: Platform (`pubsubplus-connector-framework`)](pubsubplus-connector-framework/apidocs/index.html):
  - The core platform module of the Micro-Integration Framework, which provides the core functionality for Micro-Integration runtime.
  - Use this for implementing the Micro-Integration application itself.
- [Micro-Integration Framework: IO Common (`pubsubplus-connector-io-common`)](pubsubplus-connector-io-common/apidocs/index.html):
  - Common IO utilities and interfaces used across the MDK.
  - Use this for implementing binders.
- [Micro-Integration Framework: Test Utilities (`pubsubplus-connector-framework-test-support-utilities`)](pubsubplus-connector-framework-test-support-utilities/apidocs/index.html):
  - Utilities for testing Micro-Integrations.
  - Use this for writing unit and integration tests for your Micro-Integration applications.
